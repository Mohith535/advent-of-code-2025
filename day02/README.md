# Day 2 – Advent of Code 2025
**Puzzle:** Gift Shop

This folder contains my solutions for **Day 2** of Advent of Code 2025.

---

## Problem Summary

Each input line contains ranges of product IDs.  
An ID is considered **invalid** if it is formed by repeating a smaller digit sequence.

### Part 1
An ID is invalid if it consists of a digit sequence repeated **exactly twice**.

### Part 2
An ID is invalid if it consists of a digit sequence repeated **two or more times**.
